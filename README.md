# Introduction
This is an example implementation of redux sample todo app in typescript and a server side implementation of koa/react

## Requirements
* [Node.js](https://nodejs.org/en/) >= 6.5.0

## Getting Started
1. Installation
* Install dependencies
        $ npm install
* Build react app
        $ npm run app-build
* Build koa server
        $ npm run build

2. Run
        $ node ./build/index.js
